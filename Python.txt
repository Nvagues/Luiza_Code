Tipos Basicos 

Inteiro - Composto por algaritmos numericos ( tanto negativo como positivo)
float - Numero quebrado ex: 1.75 / 60.0

Não pode utilizar virgula 

Boleano - Ou verdade ou falso 

Na logica pode ser considerada como 0 ou 1


String - Palavras, frases ou textos colocar entre aspas

nome = 'Alyce' - string 
idade = 13 - int
altura = 1.75 - float
feriado = true - bool 

Variavel 

é um nome que se refere a um valor,um comando de atribuição 
cria uma nova variavel e lhe da um valor 
São usadas para guardamos valores que serão reutilizados 
mais tarde no programa

Lista - é uma coleção ordenada de valores
cada valor e indentificada por indice
dentro de colchetes, separada por virgula 

pode criar lista com varios itens e diferentes tipos
lista vazia

compreensão de lista (loop) 

tuplas são imutaveis - não da para add reover aou alterar seus elementos

Não é feita na estrutura da Tupla apenas no conteudo desse objeto
